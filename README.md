# Credit Risk Analysis Dashboard

## Project Overview
This project analyzes credit risk using loan customer data and visualizes insights using Power BI.

## Tools Used
SQL
Power BI
Data Analysis

## Key Metrics
- Total Customers
- Bad Customers
- Default Rate
- Loss Percentage

## Key Insights
- Large loans have higher default rates
- Youth customers show higher default probability
- Certain loan purposes show higher risk
